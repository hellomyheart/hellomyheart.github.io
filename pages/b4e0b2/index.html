    do while循环

    在Java中，while循环是先判断循环条件，再执行循环。而另一种do while循环则是先执行循环，再判断条件，条件满足时继续循环，条件不满足时退出。它的用法是：

    do {
    执行循环语句
} while (条件表达式);
    1
    2
    3

    可见，do while循环会至少循环一次。

    我们把对1到100的求和用do while循环改写一下：

    public class Main {
    public static void main(String[] args) {
        int sum = 0;
        int n = 1;
        do {
            sum = sum + n;
            n ++;
        } while (n <= 100);
        System.out.println(sum);
    }
}

    使用do while循环时，同样要注意循环条件的判断。

    如果需要使用循环，while和do while要怎么选呢？

    首先while和do while可以相互转化。（有时候有些麻烦）

    对于那种从一开始就需要先执行一次，根据执行结果来决定是否要继续循环。使用do while方便点。

    对于循环还没开始，就有一个参数可以 确定是否需要循环，使用while循环比较好。

    细品下面的情景：

    考试不及格，重考，还不及格，还重考（先考一次，不及格重考，及格就不重考了）

    因为某种疫情，对于发烧的人群，需要居家隔离2周，还有问题，还需要隔离2周。不发烧的人不需要隔离。（一次也不隔离，虚拟场景，对于当下，不管是否发烧，依照相关规定，该怎么办就怎么办，为自己，为家人，为社会。）
